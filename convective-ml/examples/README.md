# examples

# Overview

Research and implementation examples for the atelier-rs engine

# General Layout

## distributed/singular_case

You can take this as a benchmark for more advanced implementations.

- Single-threaded synthetic Orderbook data generation.
- Single-threaded orderbook features generation.
- Single-threaded batch training using gradient descent.
